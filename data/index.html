Dynamic Power Regulation

Use one config line per external Modbus TCP value. The value is read and shown in the Status column.

ip IP address of the external device.
port Modbus TCP port, normally 502.
fc Function code: 3 = Holding Registers, 4 = Input Registers.
reg Start register to read.
type Data type: float, u16, s16, u32 or s32.
scale Multiplier for the read value. Example: W to kW = 0.001.
mode Planned regulation mode. max_kw will limit this charger to the remaining power.
limit Planned total power limit in kW for max_kw mode.
min Planned minimum charging power in kW. Below this value charging can be paused.
Read total power from another InnoCharge: ip=192.168.0.211;port=502;fc=3;reg=1018;type=float;scale=0.001;mode=max_kw;limit=22;min=10
Read unsigned 16-bit power value in W: ip=192.168.0.92;port=502;fc=3;reg=0;type=u16;scale=0.001;mode=max_kw;limit=22;min=10
InnoCharge power registers: L1=1012, L2=1014, L3=1016, Total=1018, Import kWh=1020, Export kWh=1022
